## How to start?

- 正常流程
1. Fork this repo.
2. 网页端Upload files.
3. Send a pull request to this repo.
4. Make sure the pull request is OK and merged.
本项目体积很大，clone所有内容到本地比较耗时，建议直接在网页端提交PR

- 若出现冲突
 1. 备份本地文件，重新Fork然后重新提交PR(不熟悉git)
 2. Rebase消除主分支冲突(熟悉git)
